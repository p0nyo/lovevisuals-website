# Love Visuals Photography

Photography portfolio website for Love Visuals, built with Next.js and Supabase.

## Tech Stack

- Next.js (frontend + backend)
- Supabase (database + storage)
- Tailwind CSS
- Framer Motion
- Lenis (smooth scroll)

## Getting Started

Install dependencies:

    npm i

Run the development server:

    npm run dev

## Project Structure

    app/          pages and routing
    components/   reusable UI components
    lib/          supabase client and utilities
    public/       static assets
